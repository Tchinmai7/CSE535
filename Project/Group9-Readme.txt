The server code is in Server/ directory.
To setup server, first do pip install -r requirements.txt
Then run download_data.py after creating a folder - "data"
Then run train.py with the path to the data folder.
Now start server as python manage.py runserver 0.0.0.0:8000

The client code is in client/ directory.
Use android studio to run it.
